# 🎓 SIETK College Website Clone

A simple and responsive front-end clone of the SIETK (Siddharth Institute of Engineering & Technology) website built using HTML and CSS.

---

## 🚀 Project Overview

This project recreates the SIETK homepage with:
- Header section
- Navigation bar
- Banner image
- Course cards section
- Hover effects
- Responsive layout using Flexbox

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
